- What is the difference between ***var*** and ***let***? let can be reassigned but not redeclared.  var can be hoisted while let creates a scope block
- What is the difference between ***var*** and ***const***? const can't be reassigned or redeclared. var can be hoisted while const creates a scope block
- What is the difference between  let and const? you can't redeclare a const, but you can a let
- What is hoisting? js compiler lifts the declaration to the top of scope it is declared in, it is simply undefined prior to be assigned a value
